# python-web-bottle-mvc-crypto-hello-world

## Description
Uses MVC to render html pages with plain text login.

| username | password |
-----------------------
| user | pass |

## Tech stack
- python
  - bottle
  - cheroot
  - beaker

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`
- [Available](http://localhost)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
